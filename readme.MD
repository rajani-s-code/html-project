# Profile Registration Form 🚀

Hi! This is my first complete HTML project where I created a detailed **Personal & Professional Profile Registration Form**.

##  Features:
- **Personal Details:** First Name, Last Name, Age, Gender dropdown, Contact, Parents' Names, and a large Address block.
- **State Selection:** Includes a complete dropdown list of all Indian States.
- **Professional Details:** Occupation status selector with conditional placeholders.

##  Tech Stack Used:
- HTML5 (Forms, Tables, Inputs)
- Git & GitHub
